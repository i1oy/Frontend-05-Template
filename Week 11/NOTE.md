# 学习笔记

## CSS 2.1总体结构

- @charset
- @import
- rules
    - @media
    - @page
    - rule

## @规则

`@media` 基于一个或多个媒体查询的结果来应用样式表的一部分;
`@keyframes` 定义动画;
`@fontface` 理论上能定义一切字体，由此衍生出来 `icon font`;

## CSS的规则

规则分为了 选择器 和 声明(key, value) 两个部分。

选择器部分目前主要关注 `Level 3` 的标准（`Level 4`处于Working Draft）；
Key部分关注`Variable`的内容；
